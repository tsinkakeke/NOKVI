# NOKVI – Crocheting & Knitting Community Platform

NOKVI is a web application dedicated to crocheting and knitting enthusiasts. The platform allows users to discover patterns, learn new techniques, explore tutorials, save favorite projects, and find inspiration for their next handmade creation. The website is designed to provide a welcoming and organized space for both beginners and experienced crafters.

Also, the name comes from Fshavian dictionary from word nokvi(/nɔkʰvi/ - ნოქვი - ), which techniacally has nothing to do with this website other than my pure satisfaction.

## Before Running the Project

Make sure to:

* Configure the API settings in `js/api.js` if external data sources are used.
* Update the `<title>` tag on all pages to **NOKVI**.
* Customize colors, fonts, and theme variables to match the NOKVI brand.
* Verify that all navigation links and page connections function correctly.

## Features

* Browse crocheting and knitting patterns
* Learn through beginner-friendly guides and tutorials
* Explore a gallery of completed projects
* Save favorite patterns and projects
* Search for inspiration and crafting ideas
* Responsive design for desktop and mobile devices
* User-friendly interface for all skill levels

NOKVI was developed as a resource and inspiration hub for crocheting and knitting enthusiasts, helping users learn new skills, discover creative projects, and organize their favorite patterns in one place.
